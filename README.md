# GuideRealm Quote Generator

A simple React app used to demonstrate how to download, clone, and run a project from GitHub.

## What it does

Click the button to get a random programming quote. That's it — built to be simple and beginner-friendly.

## Tech stack

This project is built with **React**. You can confirm this by opening `package.json` and checking the `dependencies` field — you'll see `react` and `react-dom` listed there, along with `vite` as the build tool under `devDependencies`.

## Requirements

- [Node.js](https://nodejs.org) installed on your system (this gives you `npm`)

## How to run it

1. Download or clone this repository.
2. Open a terminal in the project folder.
3. Install dependencies:

   ```
   npm install
   ```

4. Start the app:

   ```
   npm start
   ```

5. Open the local URL shown in your terminal (usually `http://localhost:5173`).
